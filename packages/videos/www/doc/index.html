I. Introduction

This document describes the requirements for Videos application. II. Vision Statement

Everybody can see videos in Youtube or in the other web site that offer you upload videos. This sentenses is true when you access your home network or your family network, but if you need to access one video in youtube when you stay in your work, it's can be not realy easy. The goal of this application is upload institucional videos in a website and it will be accessible for everyone.

III. System/Application Overview

The Videos application will consist primarily of a user interface that allows individuals to upload your videos files and, the public, see it in a web browser with a player.

IV. Use Case and User Scenarios

Using Videos in subsite

The government create one web site to publicity a program content. The kind of content that they have include a lot of videos and they need to show for everyone. Upload to Youtube don't work, because the people at the work can't access Youtube.com. So, to resolve this problem, they need to create one instance of videos package and upload their videos.
